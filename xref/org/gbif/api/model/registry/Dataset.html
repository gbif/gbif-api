56  /**
57   * A GBIF dataset which provides occurrence data, checklist data, sampling event data or metadata.
58   * This Dataset class is covering all the GBIF metadata profile v1.3, but only a few properties are kept in the
59   * database table:
60   * <ul>
61   * <li>key</li>
62   * <li>parentDatasetKey</li>
63   * <li>duplicateOfDatasetKey</li>
64   * <li>version</li>
65   * <li>installationKey</li>
66   * <li>publishingOrganizationKey</li>
67   * <li>publishingOrganizationName</li>
68   * <li>networkKeys</li>
69   * <li>license</li>
70   * <li>maintenanceUpdateFrequency</li>
71   * <li>external</li>
72   * <li>numConstituents</li>
73   * <li>type</li>
74   * <li>subtype</li>
75   * <li>title</li>
76   * <li>alias</li>
77   * <li>abbreviation</li>
78   * <li>description</li>
79   * <li>language</li>
